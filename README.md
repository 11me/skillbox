# skillbox

Universal AI agent skills for Go, Kubernetes, and GitOps development.

## Skills

| Skill | Description |
|-------|-------------|
| [go-development](go-development/) | Go patterns: services, repositories, handlers, testing |
| [helm-chart-developer](helm-chart-developer/) | Helm chart authoring, ESO integration |
| [flux-gitops](flux-gitops/) | Flux CD scaffolding, multi-environment repos |

## Installation

```bash
git clone https://github.com/11me/skillbox.git
```

## Format

Skills follow [Agent Skills Specification](https://agentskills.io/specification).
